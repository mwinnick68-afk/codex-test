# codex-test

Starter Python project skeleton with a `src/` layout and tests.

## Prerequisites

- Python 3.10+

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -U pip
pip install -e ".[dev]"
```

## Run

```bash
python -m codex_test.main
```

## Test

```bash
pytest
```
